You can install plugins on Obsidian, like Excalidraw

December 4, 2025

Obsidian supports community plugins. Excalidraw adds sketching and diagramming directly into your notes.

Installing plugins:

  1. Open Settings in Obsidian
  2. Navigate to Community plugins
  3. Click Browse to search available plugins
  4. Search for the plugin you want (e.g., “Excalidraw”)
  5. Click Install, then Enable

What Excalidraw offers:

Excalidraw is a sketching tool that lets you create visual content within Obsidian:

  • Quick sketches integrated into notes
  • User interface mockups and wireframes
  • Simple charts and diagrams
  • Mindmaps using templates
  • Image annotations on canvas
  • Embedded drawings that transclude into other notes
  • Programmatic generation using ExcalidrawAutomate library

Why use plugins:

Obsidian’s plugin ecosystem transforms it from a markdown editor into a full productivity suite. Popular plugins include:

  • Excalidraw - Drawing and diagramming
  • Dataview - Query and display note data
  • Kanban - Task boards
  • Calendar - Daily note navigation
  • Templater - Advanced template scripting

Plugins automatically update embedded references when you rename drawings.
